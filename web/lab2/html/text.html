Методы func2 возвращает результат вычисления y

def func2(x)
y = x
y = x + 5 if x < 10
y = x * 5 if x > 15
return y
end

a = func2(10) # вызов метода

Определения плотностей заряда:

линейная плотность заряда, измеряется в Кл/м;
поверхностная плотность заряда, измеряется в Кл/м2;
объемная плотность заряда, измеряется в Кл/м3.
